\doxysubsection{Detailed Description}
The \doxylink{class_leaf}{Leaf} class represents the end objects of a composition. A leaf can\textquotesingle{}t have any children.

Usually, it\textquotesingle{}s the \doxylink{class_leaf}{Leaf} objects that do the actual work, whereas \doxylink{class_composite}{Composite} objects only delegate to their sub-\/components.
