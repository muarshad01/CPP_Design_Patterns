\doxysubsection{Detailed Description}
The \doxylink{class_director}{Director} is only responsible for executing the building steps in a particular sequence. It is helpful when producing products according to a specific order or configuration. Strictly speaking, the \doxylink{class_director}{Director} class is optional, since the client can control builders directly.
