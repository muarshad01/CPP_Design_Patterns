\doxysubsection{Detailed Description}
The \doxylink{class_subject}{Subject} owns some important state and notifies observers when the state changes.

The \doxylink{class_subject}{Subject} interface declares common operations for both \doxylink{class_real_subject}{Real\+Subject} and the \doxylink{class_proxy}{Proxy}. As long as the client works with \doxylink{class_real_subject}{Real\+Subject} using this interface, you\textquotesingle{}ll be able to pass it a proxy instead of a real subject.
