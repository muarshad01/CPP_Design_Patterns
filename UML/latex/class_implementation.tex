\doxysubsection{Detailed Description}
The \doxylink{class_implementation}{Implementation} defines the interface for all implementation classes. It doesn\textquotesingle{}t have to match the \doxylink{class_abstraction}{Abstraction}\textquotesingle{}s interface. In fact, the two interfaces can be entirely different. Typically the \doxylink{class_implementation}{Implementation} interface provides only primitive operations, while the \doxylink{class_abstraction}{Abstraction} defines higher-\/ level operations based on those primitives.
