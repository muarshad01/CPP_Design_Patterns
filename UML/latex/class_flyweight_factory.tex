\doxysubsection{Detailed Description}
The \doxylink{class_flyweight}{Flyweight} Factory creates and manages the \doxylink{class_flyweight}{Flyweight} objects. It ensures that flyweights are shared correctly. When the client requests a flyweight, the factory either returns an existing instance or creates a new one, if it doesn\textquotesingle{}t exist yet.
