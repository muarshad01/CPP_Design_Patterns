\doxysubsection{Detailed Description}
The \doxylink{class_caretaker}{Caretaker} doesn\textquotesingle{}t depend on the Concrete \doxylink{class_memento}{Memento} class. Therefore, it doesn\textquotesingle{}t have access to the originator\textquotesingle{}s state, stored inside the memento. It works with all mementos via the base \doxylink{class_memento}{Memento} interface.
