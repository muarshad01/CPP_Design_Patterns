\doxysubsection{Detailed Description}
The \doxylink{class_facade}{Facade} class provides a simple interface to the complex logic of one or several subsystems. The \doxylink{class_facade}{Facade} delegates the client requests to the appropriate objects within the subsystem. The \doxylink{class_facade}{Facade} is also responsible for managing their lifecycle. All of this shields the client from the undesired complexity of the subsystem.
