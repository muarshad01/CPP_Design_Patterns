\doxysubsection{Detailed Description}
Concrete Visitors implement several versions of the same algorithm, which can work with all concrete component classes.

You can experience the biggest benefit of the \doxylink{class_visitor}{Visitor} pattern when using it with a complex object structure, such as a \doxylink{class_composite}{Composite} tree. In this case, it might be helpful to store some intermediate state of the algorithm while executing visitor\textquotesingle{}s methods over various objects of the structure.
