\doxysubsection{Detailed Description}
The \doxylink{class_component}{Component} interface declares an {\ttfamily accept} method that should take the base visitor interface as an argument.

The base \doxylink{class_component}{Component} class declares common operations for both simple and complex objects of a composition.

The base \doxylink{class_component}{Component} interface defines operations that can be altered by decorators.
