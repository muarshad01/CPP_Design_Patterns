\doxysubsection{Detailed Description}
The \doxylink{class_creator}{Creator} class declares the factory method that is supposed to return an object of a \doxylink{class_product}{Product} class. The \doxylink{class_creator}{Creator}\textquotesingle{}s subclasses usually provide the implementation of this method.
